## 自动更新配置（需手动完成）

### 1. 部署更新检查函数

```bash
cd g:\Trae项目\缺货统计系统\supabase
npx supabase functions deploy check-update
```

### 2. 修改更新地址

编辑 `electron-main.js`：
```javascript
const UPDATE_CHECK_URL = '你的 Supabase Edge Function URL';
```

编辑 `supabase/functions/check-update/index.ts`：
```typescript
const UPDATE_URL = '你的 CDN 地址/updates/';
```

### 3. 发布新版本

每次发布新版本时：
1. 修改 `package.json` 版本号
2. 打包：`npm run build:win`
3. 上传 exe 到 CDN
4. 部署 check-update 函数

详细说明见 `自动更新部署指南.md`
